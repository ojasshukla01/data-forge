# Docs

## Hero / project logo

Place the Data Forge hero/logo image as `hero.png` in this folder to display it in the main README:

```
docs/hero.png
```

The image should be an isometric illustration (recommended 800px width) showing the Data Forge pipeline—data flow, processing layers, and outputs. The main README uses it as the project logo with alt text "Data Forge — Time-Aware Synthetic Data Platform".
