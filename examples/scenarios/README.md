# Example scenarios

Import these JSON files via the UI (**Scenarios → Import scenario**) or via API (`POST /api/scenarios/import` with the JSON body).

| File | Description |
|------|-------------|
| `ecommerce_quick_start.json` | Standard e-commerce generation (1000 rows). |
| `saas_billing_benchmark.json` | SaaS billing at 5k scale for benchmark runs. |
| `fintech_pipeline.json` | Fintech pack for pipeline simulation / event streams. |

After importing, you can run a scenario from the scenario detail page or load it in **Create (wizard)** or **Advanced config** to modify and run.
